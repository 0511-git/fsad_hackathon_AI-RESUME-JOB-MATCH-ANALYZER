# AI Resume Job Match Analyzer

## Description
This project analyzes resumes and compares them with job descriptions to generate a match score. It helps users understand how well their resume fits a job role.

## Features
- Upload resume
- Extract skills
- Match with job description
- Generate match score

## Tech Stack
- React (Vite)

## How to Run
npm install
npm run dev

## Output
Displays match score based on resume and job description.
